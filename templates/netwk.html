Network Data Structure

Technologies: Python, Numpy, Web Services, Pandas, NetworkX, CircosPlot, Anaconda

The following Network Data Structures can be demonstrated in: Anaconda / Jupyter Lab
[ bio ] /python_pro.../networkx_chord.ipynb

  1. User selects a protein list.
  2. A web service (String: Protein-Protein Interaction Networks) is consumed and a string is returned
  3. The string is split into a Python list data structure
  4. The list is added to a Pandas Data Frame
  5. The Pandas Data Frame scores interactions
  6. A NetworkX Graph Data Structure is instantiated
  7. Loop through the interactions caste as a NumPy Array
  8. More processing (egs: weight the edges, label & color the nodes)
  9. Create a CircosPlot Diagram
